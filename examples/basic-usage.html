AWS Policy Generator Web Component

Basic Usage

Simple usage with default settings:

<aws-policy-generator></aws-policy-generator>

With ARN List

Pre-populate ARN options for users to select from:

<aws-policy-generator arn-list='["arn:aws:s3:::my-bucket", "arn:aws:s3:::my-bucket/*", "arn:aws:lambda:us-east-1:123456789012:function:my-function"]'> </aws-policy-generator>

Dark Theme

Using dark theme with custom CSS variables:

<aws-policy-generator theme="dark" style="--aws-background-color: #1f2937; --aws-text-color: #f9fafb; --aws-border-color: #374151;"> </aws-policy-generator>

Read-only Mode

Display-only mode for showing existing policies:

<aws-policy-generator readonly></aws-policy-generator>

Event Handling

Listen to component events:

const generator = document.querySelector('aws-policy-generator'); generator.addEventListener('resource-change', (e) => { console.log('Resource changed:', e.detail); }); generator.addEventListener('action-change', (e) => { console.log('Action changed:', e.detail); }); generator.addEventListener('policy-copied', (e) => { console.log('Policy copied to clipboard'); });

Custom Styling

Customize appearance with CSS variables:

aws-policy-generator { --aws-primary-color: #0066cc; --aws-secondary-color: #ff6600; --aws-background-color: #ffffff; --aws-border-color: #dddddd; --aws-text-color: #333333; --aws-border-radius: 12px; --font-family: 'Inter', sans-serif; }
